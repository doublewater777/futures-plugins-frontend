# Futures Dev

期货团队 Claude Code 技能集。

## 包含 Plugins

| Plugin | 用途 |
|--------|------|
| dev-plan | 前端开发方案模板 |
| code-format | 代码格式化规范 |
| weblog-sdk | 埋点上报 SDK |
| monitor-sdk | 前端监控 SDK |

## 使用方式

```bash
claude plugin add https://github.com/doublewater777/futures-dev
```

Claude Code 会根据 description 自动触发相应技能。
