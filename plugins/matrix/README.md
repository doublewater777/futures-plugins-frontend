# Matrix Plugin

@fu/matrix 工具库的 Claude Code Skill 集成，提供同花顺/期货应用开发辅助功能。

## 功能

- 字符串转换（驼峰/下划线）
- 数字格式化
- 平台/系统检测
- 应用环境检测
- 存储管理
- 性能优化（debounce/throttle）
- App 桥接（生命周期、页面跳转、登录认证）

## 使用方式

在 Claude Code 中触发 skill：
- 询问 Matrix 函数用法
- 需要环境检测、格式化、存储等功能时
- 询问可用 API 函数
