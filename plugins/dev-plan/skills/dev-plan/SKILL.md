# 前端开发方案模板

当用户需要编写前端开发方案、技术方案或设计文档时使用此 skill。

## 使用方法

1. 用户触发此 skill 时，根据需求场景选择合适的模板部分
2. 从 `assets/template.md` 复制模板内容到用户文档
3. 引导用户填写各部分内容

## 模板结构

- **项目规划**: 背景目标、人员清单、人天评估
- **整体设计**: 功能实现思路
- **前端设计**: 功能、代码、容错、监控、技术难点
- **部署方案**: 发布、灰度、回滚、降级
- **第三方依赖**: 外部接口、组件、SDK
- **风险评估**: 风险识别和应对策略

## 触发场景

- 用户说"写一个开发方案"
- 用户说"帮我做个技术方案"
- 用户说"需要设计文档模板"
- 用户提到项目规划、接口设计等技术文档
